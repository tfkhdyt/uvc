# x265-converter-termux
Skrip untuk mengonversi video ke codec x265 dengan mudah dengan Termux.
Dengan codec x265, dapat membuat size videomu berkurang sampai Â±90% tanpa berkurang kualitas

## Instalasi
```Shell
# cloning source code
git clone https://github.com/tfkhdyt/x265

# buka folder x265
cd x265

# jalankan installer
sh install.sh

# ketikkan alias untuk memulai script
x265
```
